A Little Lesson on Security

Browsing the web is usually harmless, but there are many ways for people to abuse the web and make it unsafe. For instance, here are some of the things that someone can gather just from Javascript.

These values are not anything to worry about, as they are needed to make sure that Javascript runs like it is supposed to. But it does show that there are ways to get information about you without you telling anyone.

Here are some tips to help you stay secure while browsing the web, with links to articles explaining more in-depth.

  • Checking that a site uses HTTPS instead of the normal HTTP is an easy way to make sure that the site is using an encrypted connection.
  • Learning the best practices for downloading is a great way to make sure that you don't accidentally download something malicious.
  • Creating strong passwords is essential to staying secure on the web. As a general rule, the longer the password, the better.
